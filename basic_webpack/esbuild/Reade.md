# 使用

- 安装 esbuild

```bash
npm install esbuild
```
* esbuild是一个基于Go语言开发的构建工具，性能非常优秀，号称是世界上最快的构建工具。
* 使用esbuild构建项目时，需要使用esbuild的API。
* 天生支持TypeScript、JSX、CSS等文件，无需配置。
* 分包、压缩、混淆、Tree Shaking等功能一应俱全。
